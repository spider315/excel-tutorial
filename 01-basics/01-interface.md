# Excel 介面介紹

## 學習目標
- 認識 Excel 的主要介面元素
- - 了解工作簿、工作表、儲存格的概念
  - - 學會基本的導覽操作
   
    - ## 1. Excel 啟動畫面
   
    - 當你開啟 Excel 時，會看到啟動畫面，可以選擇：
    - - **空白活頁簿**：建立全新的工作簿
      - - **範本**：使用預設的範本快速開始
       
        - ## 2. 主要介面元素
       
        - ### 2.1 功能區 (Ribbon)
        - 功能區位於視窗上方，包含多個索引標籤：
       
        - | 索引標籤 | 功能說明 |
        - |---------|---------|
        - | 常用 | 剪貼簿、字型、對齊方式、數值格式 |
        - | 插入 | 表格、圖表、圖片、連結 |
        - | 版面配置 | 頁面設定、列印範圍 |
        - | 公式 | 函數插入、名稱管理 |
        - | 資料 | 排序、篩選、資料驗證 |
        - | 校閱 | 拼字檢查、保護工作表 |
        - | 檢視 | 檢視模式、凍結窗格 |
       
        - ### 2.2 名稱方塊與資料編輯列
        - - **名稱方塊**：顯示目前選取的儲存格位址（如 A1, B5）
          - - **資料編輯列**：顯示並編輯儲存格內容
           
            - ### 2.3 工作表區域
            - - **欄 (Column)**：以英文字母標示（A, B, C...）
              - - **列 (Row)**：以數字標示（1, 2, 3...）
                - - **儲存格 (Cell)**：欄與列交叉的位置（如 A1）
                 
                  - ### 2.4 工作表標籤
                  - 位於視窗下方，可以：
                  - - 切換不同的工作表
                    - - 新增工作表（點擊 `+` 按鈕）
                      - - 重新命名工作表（雙擊標籤名稱）
                       
                        - ## 3. 基本導覽操作
                       
                        - ### 鍵盤快捷鍵
                       
                        - | 快捷鍵 | 功能 |
                        - |--------|------|
                        - | `Ctrl + Home` | 移到 A1 儲存格 |
                        - | `Ctrl + End` | 移到最後一個使用的儲存格 |
                        - | `Ctrl + →` | 移到該列最後一個有資料的儲存格 |
                        - | `Ctrl + ↓` | 移到該欄最後一個有資料的儲存格 |
                        - | `Ctrl + Page Up` | 切換到上一個工作表 |
                        - | `Ctrl + Page Down` | 切換到下一個工作表 |
                        - | `Tab` | 移到右邊的儲存格 |
                        - | `Enter` | 移到下方的儲存格 |
                       
                        - ## 練習題
                       
                        - 1. 開啟 Excel，建立一個新的空白活頁簿
                          2. 2. 找到並點擊「公式」索引標籤，觀察有哪些功能
                             3. 3. 在 A1 儲存格輸入你的名字，然後按 Tab 鍵移到 B1
                                4. 4. 嘗試使用 `Ctrl + Home` 回到 A1 儲存格
                                   5. 5. 在工作表標籤上雙擊，將工作表重新命名為「練習」
