# 基本公式：SUM, AVERAGE, COUNT

## 學習目標
- 了解 Excel 公式的基本語法
- - 學會使用 SUM、AVERAGE、COUNT 等常用函數
  - - 掌握儲存格參照的方式
   
    - ## 1. 公式基礎
   
    - ### 1.1 公式的基本規則
    - - 所有公式都以 `=` 開頭
      - - 可以使用算術運算符：`+`（加）、`-`（減）、`*`（乘）、`/`（除）
        - - 可以參照其他儲存格的值
         
          - ### 1.2 簡單運算範例
         
          - ```
            =A1+B1        加法
            =A1-B1        減法
            =A1*B1        乘法
            =A1/B1        除法
            =A1^2         次方（A1 的平方）
            ```

            ## 2. SUM 函數（加總）

            ### 語法
            ```
            =SUM(數值1, 數值2, ...)
            =SUM(範圍)
            ```

            ### 範例
            | | A | B | C |
            |---|---|---|---|
            | 1 | 品項 | 金額 | |
            | 2 | 蘋果 | 30 | |
            | 3 | 香蕉 | 25 | |
            | 4 | 橘子 | 40 | |
            | 5 | **合計** | `=SUM(B2:B4)` | 結果：95 |

            ### 常用變化
            ```
            =SUM(B2:B4)          加總 B2 到 B4
            =SUM(B2,B4,B6)       加總指定儲存格
            =SUM(B:B)            加總整個 B 欄
            =SUMIF(A2:A4,"蘋果",B2:B4)  條件加總
            ```

            ## 3. AVERAGE 函數（平均值）

            ### 語法
            ```
            =AVERAGE(數值1, 數值2, ...)
            =AVERAGE(範圍)
            ```

            ### 範例
            | | A | B |
            |---|---|---|
            | 1 | 科目 | 成績 |
            | 2 | 國文 | 85 |
            | 3 | 英文 | 92 |
            | 4 | 數學 | 78 |
            | 5 | **平均** | `=AVERAGE(B2:B4)` → 85 |

            ## 4. COUNT 系列函數

            ### COUNT - 計算數值個數
            ```
            =COUNT(B2:B10)     計算範圍中有多少數值
            ```

            ### COUNTA - 計算非空白儲存格
            ```
            =COUNTA(A2:A10)    計算範圍中有多少非空白儲存格
            ```

            ### COUNTBLANK - 計算空白儲存格
            ```
            =COUNTBLANK(A2:A10)  計算範圍中有多少空白儲存格
            ```

            ### COUNTIF - 條件計數
            ```
            =COUNTIF(B2:B10,">=60")    計算及格人數
            =COUNTIF(A2:A10,"蘋果")     計算「蘋果」出現次數
            ```

            ## 5. 其他常用基本函數

            | 函數 | 用途 | 範例 |
            |------|------|------|
            | MAX | 最大值 | `=MAX(B2:B10)` |
            | MIN | 最小值 | `=MIN(B2:B10)` |
            | ROUND | 四捨五入 | `=ROUND(3.456, 2)` → 3.46 |
            | ABS | 絕對值 | `=ABS(-5)` → 5 |
            | MOD | 餘數 | `=MOD(10,3)` → 1 |

            ## 6. 儲存格參照

            ### 相對參照
            ```
            =A1    當公式複製到其他位置時，參照會自動調整
            ```

            ### 絕對參照
            ```
            =$A$1   無論複製到哪裡，永遠參照 A1
            ```

            ### 混合參照
            ```
            =$A1   欄固定，列會變
            =A$1   列固定，欄會變
            ```
