# 02 — Prompt 範本大全：Excel 圖表自動化指令集

> 本篇收錄各種圖表場景的 Prompt 範本，可直接複製修改使用。

---

## 一、Prompt 結構公式

所有圖表 Prompt 都遵循相同的結構：

```
[任務描述] + [資料來源] + [圖表規格] + [格式要求] + [輸出位置]
```

展開後：

```
請用 [圖表類型] 呈現 [什麼資料] 的 [什麼分析]。
- 資料來源：[檔案名稱]
- 資料處理：[如何加總/分組/篩選]
- X 軸：[欄位名稱]（[格式]）
- Y 軸：[欄位名稱]（[格式]）
- 系列：[分組依據]，每組一個 [線/柱/扇區]
- 配色：[顏色列表]
- 標題：「[圖表標題]」
- 大小：寬 [N]、高 [N]
- 位置：放在 [儲存格]
- 資料標籤：[顯示什麼]
```

## 二、基礎圖表 Prompt 範本

### 範本 A：基本折線圖

```
請用折線圖顯示各月份的總營收趨勢。

資料來源：raw/monthly_sales_detail.xlsx
處理方式：依「年月」加總所有區域的「營收」欄位

圖表規格：
- 類型：折線圖（LineChart）
- X 軸：月份（2025-01 到 2025-12）
- Y 軸：營收總額，格式 #,##0
- 單一系列，深藍色 #2F5496
- 折線寬度 25000 EMU
- 標題：「2025 年月度總營收趨勢」
- 大小：寬 25、高 14
- 位置：A15
```

### 範本 B：基本柱狀圖

```
請用柱狀圖比較各產品類別的年度銷售總額。

資料來源：raw/monthly_sales_detail.xlsx
處理方式：依「產品類別」加總全年「營收」

圖表規格：
- 類型：群組柱狀圖（BarChart type="col" grouping="clustered"）
- X 軸：產品類別
- Y 軸：營收，格式 #,##0
- 每根柱子用不同顏色
- 標題：「各產品年度銷售營收」
- 顯示數值資料標籤
- 大小：寬 22、高 14
- 位置：A8
```

### 範本 C：基本圓餅圖

```
請用圓餅圖顯示市場份額分佈。

資料來源：raw/market_share.xlsx
使用欄位：品牌（類別）、市占率(%)（數值）

圖表規格：
- 類型：圓餅圖（PieChart）
- 顯示：類別名稱 + 百分比
- 不顯示數值
- 自訂顏色（依序）：#2F5496, #C00000, #548235, #BF8F00, #7030A0, #A5A5A5
- 標題：「市場份額分佈」
- 大小：寬 18、高 14
- 位置：A10
```

## 三、進階圖表 Prompt 範本

### 範本 D：組合圖（柱狀 + 折線）

```
請用組合圖同時顯示預算金額、實際金額與差異率。

資料來源：raw/budget_vs_actual.xlsx

圖表規格：
- 主軸（左 Y 軸）：柱狀圖
  - 系列 1：預算金額，淺藍 #B4C6E7
  - 系列 2：實際金額，深藍 #2F5496
  - 格式：千分位 #,##0
- 副軸（右 Y 軸）：折線圖
  - 系列 3：差異率(%)，暗紅色 #C00000 折線
  - 折線寬度 25000 EMU
  - 副軸 axId = 200
- X 軸：月份
- 標題：「月度預算 vs 實際金額」
- 大小：寬 28、高 15
- 位置：A16

技術提示：
1. 先建立 BarChart 加入柱狀系列
2. 再建立 LineChart 加入折線系列
3. 設定 line_chart.y_axis.axId = 200
4. 設定 bar_chart.y_axis.crosses = "min"
5. 用 bar_chart += line_chart 合併
```

### 範本 E：雷達圖

```
請用雷達圖比較各區域的客戶滿意度表現。

資料來源：raw/customer_survey.xlsx

圖表規格：
- 類型：雷達圖（RadarChart type="marker"）
- 5 個維度軸：產品品質、售後服務、價格合理性、交貨速度、技術支援
- 4 條線：北區、中區、南區、東區（各用不同顏色）
- 線寬 25000 EMU
- 標題：「各區域客戶滿意度雷達圖」
- 大小：寬 22、高 16
- 位置：A8

技術提示：
雷達圖需要逐列建立 Series 物件，不能用 add_data() 一次加入。
每個區域是一個 Series，5 個維度是 categories。
```

### 範本 F：散佈圖

```
請用散佈圖分析產品營收與毛利率的關係。

資料來源：raw/monthly_sales_detail.xlsx
處理方式：
1. 依「產品類別」加總「營收」和「毛利」
2. 計算毛利率 = 毛利 / 營收 × 100

圖表規格：
- 類型：散佈圖（ScatterChart）
- X 軸：總營收，格式 #,##0
- Y 軸：毛利率(%)，格式 0.0"%"
- 資料點不連線（noFill on line）
- 顯示 Y 軸值資料標籤
- 標題：「產品營收 vs 毛利率分析」
- 大小：寬 24、高 15
- 位置：A9
```

## 四、多圖表整合 Prompt 範本

### 範本 G：單一檔案多圖表

```
請建立一個 Excel 檔案，包含 4 個工作表，每個工作表有資料表和圖表：

工作表 1「月度趨勢」：
- 資料：各區域月度營收 pivot 表
- 圖表：折線圖，放在 A15

工作表 2「市占率」：
- 資料：品牌市占率表
- 圖表：圓餅圖，放在 A10

工作表 3「預算對比」：
- 資料：月度預算 vs 實際
- 圖表：組合圖（柱狀+折線），放在 A16

工作表 4「績效排名」：
- 資料：業務員業績表
- 圖表：水平長條圖，放在 E1

每個工作表的表頭格式：粗體白字、深色底、置中、細框線。
```

## 五、格式控制 Prompt 範本

### 範本 H：完整格式規範

```
所有報表的共用格式規範：

表頭樣式：
- 字體：微軟正黑體、11pt、粗體、白色
- 背景：依工作表性質選擇
  - 財務相關 = 深藍 #2F5496
  - 績效相關 = 深綠 #548235
  - 分析相關 = 深金 #BF8F00
- 對齊：水平置中、垂直置中、自動換行

資料儲存格：
- 邊框：上下左右細框線
- 對齊：水平置中、垂直置中
- 數字格式：
  - 金額 → #,##0
  - 百分比 → 0.0%
  - 日期 → YYYY-MM

欄寬：
- 自動調整，最大寬度 30 個字元
```

### 範本 I：條件式格式

```
在業務員績效表中加入條件式格式：

達成率欄位：
- ≥ 100%：綠色底 #C6EFCE + 深綠字 #006100
- 80% - 99%：黃色底 #FFEB9C + 深黃字 #9C6500
- < 80%：紅色底 #FFC7CE + 深紅字 #9C0006

實際業績欄位：
- 加入藍色資料條 (DataBar)，顏色 #5B9BD5
```

## 六、除錯用 Prompt 範本

### 範本 J：驗證圖表

```
請檢查 output/ 資料夾中所有 Excel 檔案的圖表是否正確：

1. 開啟每個 .xlsx 檔案
2. 檢查每個工作表是否包含圖表物件
3. 列出每張圖表的：
   - 圖表類型
   - 標題
   - 資料系列數量
   - X 軸類別數量
4. 如果有圖表缺失或資料為空，報告問題
```

### 範本 K：修復圖表

```
圖表 05_budget_vs_actual_combo.xlsx 的折線被壓在底部看不到。
請檢查以下項目並修復：

1. 折線是否有設定副軸（axId = 200）？
2. 柱狀圖的 Y 軸是否設定 crosses = "min"？
3. 副軸的數值範圍是否合理？
4. 如果以上都正確，請確認資料引用範圍是否包含差異率欄位。
```

## 七、場景化 Prompt 範例

### 場景 1：月報自動化

```
我每月需要製作部門月報，包含以下圖表：
1. 本月各產品線營收柱狀圖
2. 近 6 個月營收趨勢折線圖
3. 費用結構圓餅圖
4. 業績目標達成率表格（含條件式格式）

請讀取 raw/monthly_report_data.xlsx，自動產生以上 4 張圖表，
整合到一個 output/monthly_report_YYYY_MM.xlsx 檔案中，
每張圖表各佔一個工作表。
```

### 場景 2：年度報告

```
年度報告需要以下視覺化圖表：
1. 12 個月營收與利潤趨勢（雙軸組合圖）
2. 各事業部營收佔比（圓餅圖）
3. 季度同比成長率（柱狀圖）
4. 客戶滿意度與 NPS 變化（折線圖）
5. 區域業績排名（水平長條圖）

每張圖表需要：
- 統一配色方案（商務藍色系）
- 中文標題 14pt 粗體
- 圖例放在圖表下方
- Y 軸千分位格式
```

### 場景 3：競品分析

```
請根據競品分析資料，產生以下圖表：
1. 市占率變化趨勢（折線圖，X 軸=季度，每家公司一條線）
2. 當前市占率分佈（圓餅圖）
3. 產品功能評比（雷達圖，5-8 個維度）
4. 價格 vs 市占率（散佈圖，點的大小代表營收）

輸出到 output/competitor_analysis.xlsx，4 個工作表。
```

## 八、Prompt 品質檢查清單

在送出圖表 Prompt 之前，逐項檢查：

- [ ] 是否指定了圖表類型（中文名 + 英文類別名）？
- [ ] 是否指定了 X 軸和 Y 軸的欄位？
- [ ] 是否指定了 Y 軸的數字格式（#,##0 或 0.0%）？
- [ ] 是否指定了配色方案（hex 色碼）？
- [ ] 是否指定了圖表標題？
- [ ] 是否指定了圖表大小（寬×高）？
- [ ] 是否指定了圖表位置（哪個儲存格）？
- [ ] 是否指定了資料標籤要顯示什麼？
- [ ] 組合圖是否有提到「副軸」？
- [ ] 圓餅圖是否有指定各扇區顏色？
