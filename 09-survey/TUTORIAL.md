# 問卷調查統計分析 — AI 指令教學

> **難度**：進階
> **適用對象**：人資人員、企劃人員、市調分析師
> **學習目標**：用 AI 統計問卷結果，產生各部門比較與交叉分析

---

## 📌 課程情境

**你是一家公司的人資專員**，剛完成年度員工滿意度調查：

1. 收到 200 份問卷填答
2. 需要計算各部門的滿意度平均
3. 找出哪些題目分數最低（需要改善）
4. 做職級 × 年資的交叉分析
5. 產生圖表給主管報告

用 Excel 手動算 COUNTIFS 和做樞紐表很花時間，現在用 AI 一次搞定。

## 🎯 Prompt 設計原則

### 原則 1：分數範圍驗證

```
❌ 不好的寫法：
修正不合理的分數

✅ 好的寫法：
分數驗證與修正：
- 有效範圍：1-5 分
- 分數 < 1：修正為 1
- 分數 > 5：修正為 5
- 空值：以該題中位數填補
```

### 原則 2：統計指標要完整

```
❌ 不好的寫法：
計算平均分數

✅ 好的寫法：
各題目統計指標：
- 平均分數：MEAN，取到小數第 2 位
- 標準差：STD，取到小數第 2 位
- 最高分：MAX
- 最低分：MIN
- 滿意比例：(分數 >= 4 的人數 / 總人數) × 100%
- 不滿意比例：(分數 <= 2 的人數 / 總人數) × 100%
```

### 原則 3：交叉分析要指定維度

```
❌ 不好的寫法：
做交叉分析

✅ 好的寫法：
樞紐表設定：
- 列（Row）：職級（依序：一般職員、資深專員、主任、經理、副總）
- 欄（Column）：年資（依序：未滿1年、1-3年、3-5年、5-10年、10年以上）
- 值（Value）：整體滿意度的平均（MEAN）
- 整體滿意度 = Q1~Q10 的平均
```

### 原則 4：圖表規格要明確

```
❌ 不好的寫法：
畫圖表

✅ 好的寫法：
部門比較柱狀圖：
- 圖表類型：直立柱狀圖（Column Chart）
- X 軸：部門名稱
- Y 軸：整體平均分數（範圍 0-5）
- 標題：「各部門整體滿意度」
- 圖表大小：15×10
- 位置：D2 儲存格
```

## 📝 完整 Prompt 範本

### Prompt 1：資料清理

```
【角色】你是資深數據分析師，熟悉問卷分析與 Python 程式設計

【情境】
- 輸入檔案：raw/survey_responses.xlsx（問卷填答結果）
- 輸出目錄：output/

【任務】資料清理

1. 分數範圍驗證（Q1-Q10，有效範圍 1-5）：
   - 分數 < 1 → 修正為 1
   - 分數 > 5 → 修正為 5
   - 空值 → 以該題中位數填補

2. 日期格式標準化（填答日期）：
   - 統一為 YYYY/MM/DD

3. 部門名稱標準化：
   - 移除多餘空白
   - 若不以「部」結尾，補上「部」

4. 職級名稱修正：
   - 「一般」→「一般職員」
   - 「經裡」→「經理」

【輸出】
- cleaned_survey.xlsx：清理後問卷
- cleaning_log.xlsx：清理日誌

【驗證】顯示清理修正的總筆數
```

## 💡 常見錯誤與修正

### 錯誤 1：空值導致平均計算錯誤

**問題**：某題有空值，平均分數變成 NaN

**原因**：numpy.mean 預設會因為 NaN 而回傳 NaN

**修正**：先填補空值（用中位數或排除計算）

### 錯誤 2：樞紐表順序錯亂

**問題**：職級順序變成字母順序（一般職員、主任、副總...）

**原因**：pandas 預設依字母排序

**修正**：使用 `.reindex()` 指定順序

### 錯誤 3：百分比格式錯誤

**問題**：滿意比例顯示 0.45 而非 45%

**原因**：沒有乘以 100

**修正**：計算後乘以 100，加上 % 符號
