# Excel SUM 函數教學

## 1. SUM 函數基本語法說明

`SUM` 是 Excel 中最常用的函數之一，用於將一組數字加總。

### 基本語法

```
=SUM(數值1, [數值2], ...)
```

| 參數 | 說明 |
|------|------|
| 數值1 | 必填。要加總的第一個數字、儲存格或範圍 |
| 數值2, ... | 選填。最多可加入 255 個參數 |

### 支援的參數類型

- **單一儲存格**：`=SUM(A1)`
- **儲存格範圍**：`=SUM(A1:A10)`
- **多個範圍**：`=SUM(A1:A5, C1:C5)`
- **直接輸入數字**：`=SUM(10, 20, 30)`
- **混合使用**：`=SUM(A1:A5, 100, C1)`

---

## 2. 基本用法範例

### 範例一：加總一欄數字

假設 A1:A5 儲存以下銷售金額：

| 儲存格 | 數值 |
|--------|------|
| A1 | 1,200 |
| A2 | 3,500 |
| A3 | 2,800 |
| A4 | 4,100 |
| A5 | 1,900 |

在 A6 輸入：

```
=SUM(A1:A5)
```

結果：**13,500**

---

### 範例二：加總一列數字

假設 B1:F1 存放每月費用：

| B1 | C1 | D1 | E1 | F1 |
|----|----|----|----|----|
| 500 | 750 | 620 | 890 | 430 |

在 G1 輸入：

```
=SUM(B1:F1)
```

結果：**3,190**

---

### 範例三：加總不連續範圍

加總 A 欄與 C 欄的數字，跳過 B 欄：

```
=SUM(A1:A10, C1:C10)
```

---

### 範例四：跨工作表加總

加總「一月」到「三月」工作表中相同儲存格（B5）的值：

```
=SUM(一月:三月!B5)
```

---

## 3. 搭配條件的進階用法

當你需要「只加總符合某條件的數字」時，請使用 `SUMIF` 或 `SUMIFS`。

---

### SUMIF — 單一條件加總

#### 語法

```
=SUMIF(條件範圍, 條件, 加總範圍)
```

#### 範例：只加總「業務部」的銷售額

| A 欄（部門） | B 欄（銷售額） |
|------------|--------------|
| 業務部 | 5,000 |
| 行銷部 | 3,200 |
| 業務部 | 7,800 |
| 行銷部 | 2,100 |
| 業務部 | 4,500 |

```
=SUMIF(A1:A5, "業務部", B1:B5)
```

結果：**17,300**（5,000 + 7,800 + 4,500）

#### 條件運算子範例

| 目的 | 公式範例 |
|------|----------|
| 大於 1000 | `=SUMIF(B1:B10, ">1000", B1:B10)` |
| 小於等於 500 | `=SUMIF(B1:B10, "<=500", B1:B10)` |
| 等於某儲存格值 | `=SUMIF(A1:A10, D1, B1:B10)` |
| 包含「台北」 | `=SUMIF(A1:A10, "*台北*", B1:B10)` |
| 不等於空白 | `=SUMIF(A1:A10, "<>", B1:B10)` |

---

### SUMIFS — 多條件加總

#### 語法

```
=SUMIFS(加總範圍, 條件範圍1, 條件1, [條件範圍2, 條件2], ...)
```

#### 範例：加總「業務部」且銷售額大於 4,000 的資料

```
=SUMIFS(B1:B5, A1:A5, "業務部", B1:B5, ">4000")
```

結果：**12,500**（5,000 + 7,800 — 4,500 未超過，不計）

> **注意**：`SUMIFS` 的第一個參數是加總範圍，與 `SUMIF` 的參數順序不同。

---

### SUMPRODUCT — 彈性多條件加總（不需要輔助欄）

適合需要動態條件或跨欄條件的情境：

```
=SUMPRODUCT((A1:A10="業務部")*(C1:C10="台北")*B1:B10)
```

意義：加總部門為「業務部」且地區為「台北」的所有銷售額。

## 4. 常見錯誤與除錯方式

### 錯誤一：`#VALUE!`

**原因**：範圍內包含文字，且 Excel 無法自動忽略。

**範例**：

```
=SUM(A1:A5)   ← 若 A3 = "無資料"（文字），SUM 通常忽略它
=SUM(A1, "abc")   ← 直接傳入文字字串，會回傳 #VALUE!
```

**解法**：
- 確認資料欄位沒有混入文字
- 使用 `=IFERROR(SUM(A1:A5), 0)` 避免錯誤顯示

---

### 錯誤二：結果為 0，但看起來有數字

**原因**：儲存格中的數字實際上是「文字格式」，SUM 無法計算。

**判斷方法**：
- 數字靠左對齊（通常為文字），靠右對齊才是數值

**解法**：

```
1. 選取該欄 → 資料 → 資料剖析 → 完成（強制轉換為數值）
2. 或在空白格輸入 1，複製後「選擇性貼上 → 乘」
3. 或使用公式：=VALUE(A1)
```

---

### 錯誤三：`#REF!`

**原因**：SUM 參照的範圍因刪除列/欄而失效。

```
=SUM(A1:A#REF!)   ← 表示參照已損毀
```

**解法**：重新選取正確的儲存格範圍。

---

### 錯誤四：忘記加總範圍與條件範圍大小不一致（SUMIF/SUMIFS）

**原因**：`SUMIF` 的條件範圍與加總範圍列數不同。

```
=SUMIF(A1:A5, "業務部", B1:B10)   ← 大小不一致，結果可能錯誤
```

**解法**：確認兩個範圍的列數（或欄數）相同。

---

### 錯誤五：使用萬用字元時無效

**原因**：萬用字元（`*`、`?`）只適用於文字條件，不能用於數字比較。

```
=SUMIF(B1:B10, "*5*", B1:B10)   ← 無法篩選包含數字 5 的數值
```

**解法**：改用 `SUMPRODUCT` 搭配文字轉換：

```
=SUMPRODUCT((ISNUMBER(FIND("5", TEXT(B1:B10,"0"))))*B1:B10)
```

---

### 快速除錯技巧

| 技巧 | 說明 |
|------|------|
| `Ctrl + ~` | 切換「顯示公式」模式，快速檢查所有公式 |
| 點選公式列中的範圍 | Excel 會用顏色框標示對應儲存格 |
| `F2` 進入編輯模式 | 可看到公式參照的儲存格 |
| `=ISNUMBER(A1)` | 確認儲存格是否為數值格式 |
| `公式稽核 → 追蹤先例` | 視覺化顯示公式的資料來源 |

## 小結

| 函數 | 用途 | 條件數 |
|------|------|--------|
| `SUM` | 直接加總範圍 | 無條件 |
| `SUMIF` | 單一條件加總 | 1 個 |
| `SUMIFS` | 多條件加總 | 2～127 個 |
| `SUMPRODUCT` | 彈性條件加總 | 不限 |

> **學習建議**：先熟悉 `SUM` 和 `SUMIF`，再進階到 `SUMIFS`。遇到複雜條件時，`SUMPRODUCT` 是強大的備用方案。
