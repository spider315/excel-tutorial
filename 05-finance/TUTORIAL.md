# 財務報表自動化 — AI 指令教學

> **難度**：進階
> **適用對象**：會計、財務人員
> **學習目標**：用 AI 自動彙整 12 個月費用明細，產生損益表與預算達成分析

---

## 📌 課程情境

**你是一家中型企業的財務專員**，每到月底和年底都要花大量時間：

1. 整理各部門傳來的費用明細
2. 把 12 個月的資料合併成年度損益摘要
3. 比對預算與實際支出，標記超支項目
4. 產生給老闆看的財務儀表板

這些工作過去要花 2-3 天，現在用 AI 只要幾分鐘。

| 日期 | 費用發生日 | 2025/01/15 |
| 部門 | 費用歸屬部門 | 業務部 |
| 科目代碼 | 會計科目代碼 | 5100 |
| 科目名稱 | 會計科目名稱 | 基本薪資 |

### 故意注入的髒資料

| 問題類型 | 範例 |
|----------|------|
| 日期格式不一致 | `2025-01-06`、`25/02/15`、`2025.04.12` |
| 金額為負數 | `-5000` |
| 科目代碼格式錯誤 | `51OO`（O 不是 0） |
| 部門名稱錯誤 | `業務`（缺少「部」） |
| 重複傳票 | 同一筆費用記兩次 |
| 核准狀態不一致 | `approved`、`核准` |

## 🎯 Prompt 設計原則

### 原則 1：明確指定輸入輸出

```
❌ 不好的寫法：
幫我整理費用資料

✅ 好的寫法：
請讀取 raw/expense_detail_2025.xlsx，清理後輸出到 output/cleaned_expense_detail.xlsx
```

### 原則 2：列舉所有清理規則

```
❌ 不好的寫法：
修正日期格式

✅ 好的寫法：
日期格式統一為 YYYY/MM/DD：
- 2025-01-06 → 2025/01/06
- 25/02/15 → 2025/02/15
- 2025.04.12 → 2025/04/12
```

### 原則 3：指定彙總維度與公式

```
❌ 不好的寫法：
幫我做月度費用彙總

✅ 好的寫法：
以「月份」為列、「科目類別」為欄，計算借方金額的 SUM，
最後一欄加上「合計」= 各類別的橫向加總
```

### 原則 4：條件格式規則要明確

```
❌ 不好的寫法：
超支的標紅色

✅ 好的寫法：
條件格式設定：
- 達成率 > 100%：儲存格背景 #FFC7CE（淺紅）
- 達成率 90-100%：儲存格背景 #FFEB9C（淺黃）
- 達成率 < 90%：儲存格背景 #C6EFCE（淺綠）
```

## 📝 完整 Prompt 範本

### Prompt 1：費用明細清理

```
【角色】你是資深財務分析師，熟悉會計作業與 Python 程式設計

【情境】
- 輸入檔案：raw/expense_detail_2025.xlsx（全年費用明細，約 400 筆）
- 輸出目錄：output/

【任務】資料清理
1. 日期格式統一為 YYYY/MM/DD
   - 2025-01-06 → 2025/01/06
   - 25/02/15 → 2025/02/15
   - 2025.04.12 → 2025/04/12

2. 金額修正
   - 負數金額轉為正數（取絕對值）

3. 科目代碼修正
   - O（英文字母）→ 0（數字）
   - 移除前後空白

4. 部門名稱標準化
   - 移除多餘空白
   - 若不以「部」結尾，則補上「部」

5. 核准狀態標準化
   - approved → 已核准
   - 核准 → 已核准

6. 移除重複傳票（依傳票編號判斷，保留第一筆）

【輸出】
- cleaned_expense_detail.xlsx：清理後的費用明細
- cleaning_log.xlsx：清理日誌（欄位、列號、原始值、修正值、原因）

【驗證】顯示清理修正的總筆數
```

### Prompt 2：損益表彙整

```
【角色】你是資深財務分析師

【情境】
- 費用明細：output/cleaned_expense_detail.xlsx
- 科目對照：raw/account_chart.xlsx
- 輸出：output/income_statement.xlsx

【任務】產生年度損益表

Sheet 1「月度費用明細」：
- 列：月份（1-12）
- 欄：各科目類別（從 account_chart.xlsx 取得）
- 值：借方金額的 SUM
- 最後一欄：合計（橫向加總）

Sheet 2「費用類別彙總」：
- 欄位：費用類別、年度金額、佔比
- 依年度金額降冪排序
- 佔比 = 該類別金額 / 總金額 × 100%

Sheet 3「部門費用分析」：
- 列：部門
- 欄：各科目類別
- 值：借方金額的 SUM
- 最後一欄：合計

【格式】
- 標題列：深藍底（#2F5496）、白字、置中
- 金額欄：千分位格式（#,##0）
- 佔比欄：一位小數 + % 符號
```

### Prompt 3：預算達成分析

```
【角色】你是資深財務分析師

【情境】
- 費用明細：output/cleaned_expense_detail.xlsx
- 科目對照：raw/account_chart.xlsx
- 年度預算：raw/annual_budget_2025.xlsx
- 輸出：output/budget_analysis.xlsx

【任務】預算 vs 實際分析

1. 合併費用明細與科目對照，取得科目類別

2. 依「部門 + 費用類別」彙總實際支出（借方金額 SUM）

3. 與年度預算表 LEFT JOIN（部門 + 費用類別）

4. 計算：
   - 達成率 = 實際支出 / 年度預算 × 100%
   - 差異 = 年度預算 - 實際支出

5. 狀態判斷：
   - 達成率 > 100%：「超支」
   - 達成率 90-100%：「警示」
   - 達成率 < 90%：「正常」

【格式】
- 標題列：深藍底、白字
- 金額欄：千分位格式
- 達成率：一位小數 + %
- 條件格式：
  - 超支 → 淺紅底 #FFC7CE
  - 警示 → 淺黃底 #FFEB9C
  - 正常 → 淺綠底 #C6EFCE

【驗證】顯示超支項目數量與警示項目數量
```

### Prompt 4：財務儀表板

```
【角色】你是資深財務分析師

【情境】
- 已完成：income_statement.xlsx、budget_analysis.xlsx
- 輸出：output/finance_dashboard.xlsx

【任務】建立財務儀表板

Sheet 1「月度趨勢」：
- 複製 income_statement 的月度費用明細
- 新增折線圖：X 軸=月份，Y 軸=合計金額
- 圖表位置：A15
- 圖表大小：15×8

Sheet 2「費用結構」：
- 複製費用類別彙總
- 可加圓餅圖顯示佔比（選配）

Sheet 3「部門預算達成」：
- 欄位：部門、年度預算（SUM）、實際支出（SUM）、達成率
- 新增群組柱狀圖：預算 vs 實際
- 圖表位置：F2
- 圖表大小：15×8

【格式】
- 統一標題列樣式
- 金額千分位
- 圖表標題：「月度費用趨勢」、「部門預算 vs 實際支出」
```

## 💡 常見錯誤與修正

### 錯誤 1：合併後欄位找不到

**問題**：科目類別欄位在合併後變成 NaN

**原因**：科目代碼格式不一致（有的有空白，有的是字母 O）

**修正**：先清理科目代碼格式，再進行合併

### 錯誤 2：金額欄被當成文字

**問題**：SUM 結果為 0 或出錯

**原因**：Excel 讀進來後金額欄是字串

**修正**：在 Prompt 中明確指定 `pd.to_numeric(errors="coerce")`

### 錯誤 3：條件格式沒有套用

**問題**：儲存格沒有變色

**原因**：openpyxl 的條件格式需要指定完整 range

**修正**：使用 PatternFill 直接賦值到特定儲存格
