請在 03-advanced/ 目錄下建立 advanced_analysis.py，
讀取 03-advanced/raw/ 的四份原始 Excel，處理結果輸出到 03-advanced/output/。

任務一：資料清理與標準化
清理規則：
1. 日期格式統一為 YYYY-MM-DD（/ → -）
2. 業務員姓名去空白、移除括號標注（如「張冠宇（代）」→「張冠宇」）
3. 產品名稱去空白、修正錯字（「智彗」→「智慧」、「藍芽」→「藍牙」）
4. 數值欄位轉為數值型態
每次修正記錄到 cleaning_log.xlsx（欄位：來源檔案、列號、欄位、原始值、修正值、問題描述）
輸出：cleaned_monthly_sales.xlsx、cleaned_budget_targets.xlsx、
      cleaned_customer_feedback.xlsx、cleaning_log.xlsx

任務二：多維度銷售分析 → sales_analysis_report.xlsx
前置：排除銷售金額 ≤ 0 或數量 ≤ 0 的紀錄

Sheet 1「月度銷售趨勢」
欄位：月份, 訂單數, 銷售總額, 平均客單價, 月成長率%
MoM% = (本月 - 上月) / 上月 × 100
圖表：折線圖(銷售總額,A15)、長條圖(訂單數,A32)

Sheet 2「部門銷售樞紐」行=部門, 列=月份, 值=SUM(銷售金額), 加「年度合計」欄

Sheet 3「業務員排名」
欄位：排名, 業務員, 部門, 訂單數, 銷售總額, 平均每單金額
按銷售總額降序，Top 3 加淺綠底 #C6EFCE

Sheet 4「產品類別分布」
欄位：產品類別, 銷售筆數, 銷售總額, 銷售佔比
圓餅圖(A10)

Sheet 5「客戶區域分布」欄位：客戶區域, 銷售筆數, 銷售總額，按總額降序

任務三：預算達成率 → kpi_dashboard.xlsx

Sheet 1「個人KPI達成率」
合併銷售實際額與預算目標（以業務員為 key）
達成率% = 實際 / 目標 × 100
狀態：≥120% ★超標, ≥100% ✔達成, ≥80% △接近, <80% ✘未達成
條件格式：≥100 綠底 #C6EFCE、80-99 黃底 #FFEB9C、<80 紅底 #FFC7CE

Sheet 2「部門KPI彙總」按部門彙總，加長條圖(A10)

任務四：產品利潤交叉分析 → product_profit_report.xlsx
銷售彙總 merge 產品目錄(成本) merge 回饋(平均滿意度)
毛利率% = (銷售總額 - 數量×成本) / 銷售總額 × 100
按毛利降序，毛利率欄加資料條(藍 #5B9BD5)

任務五：資料品質報告 → data_quality_report.xlsx
Sheet 1 問題清單：負數金額、數量0、重複訂單、評分超範圍、預算Q合計超115%
Sheet 2 問題統計：按問題類型×來源檔案
Sheet 3 清理日誌：來自任務一的修正紀錄

所有報表格式：
- 標題列：粗體白字、深藍底(#2F5496)、置中自動換行
- 細邊框、垂直置中、欄寬自動調整（中文字算2字寬，10-30範圍）
- 可用不同標題色區分（藍 #2F5496、綠 #548235、橘 #BF8F00）

完成後列出 output 所有檔案及大小，顯示關鍵指標摘要。
