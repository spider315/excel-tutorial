                        4. ## 如何貢獻
                       
                        5. 本專案使用 **Claude Code GitHub Actions** 來自動產生教學內容：
                       
                        6. 1. 前往 [Issues](../../issues) 頁面
                           2. 2. 建立新 Issue，描述你想要的教學內容
                              3. 3. 為 Issue 加上 `claude` 標籤
                                 4. 4. Claude 會自動讀取 Issue 並建立 PR 來新增教學內容
